Replace “search + browse” with AI chat based discovery for a paid video library

Role: AI Product Manager (AI Product)

Duration: 2026

Description: InVideo AI Assistant is a conversational discovery and growth intelligence layer built for a paid video learning ecosystem. Instead of relying on traditional search and browsing, the product uses retrieval augmented generation, semantic intent understanding, and catalog grounded recommendations to guide users from problem to playback within a single chat interaction. The system ensures trust and relevance by restricting all recommendations to database verified content, delivering actionable video cards, follow up guidance, and seamless playback. Architecturally, the solution combines vector search using pgvector, Supabase managed embeddings, Gemini powered reasoning, and strict JSON response contracts to ensure reliability, observability, and scalable discovery as the catalog grows. The design prioritizes grounding, low friction user journeys, and measurable chat to play conversion, positioning the assistant as the primary interface for both content discovery and creator growth.
